# 📦 Inventory Management System

A simple terminal-based Inventory Management System built with **Python**. This application allows users to efficiently manage inventory records through an interactive command-line interface. It is designed to demonstrate core Python programming concepts such as file handling, modular programming, and data management.

---

## ✨ Features

- 📦 Product Management
  - Add new products
  - View product catalog
  - Update product details
  - Remove products

- 📊 Inventory Management
  - Track available stock
  - Update inventory quantities
  - Monitor product availability

- 🧾 Billing System
  - Generate customer bills
  - Calculate total purchase amount
  - Display bill summary

- 📈 Reports
  - View inventory reports
  - Display product information
  - Access billing-related summaries

- 💾 Persistent data storage using local files
- 🖥️ Interactive menu-driven command-line interface

## 🛠️ Tech Stack

- Python 3
- File Handling
- Command-Line Interface (CLI)
- Functional Programming Approach

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or above installed on your system.

Check your Python installation:

```bash
python --version
```

or

```bash
python3 --version
```

---

## ▶️ Running the Project Locally

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/Inventory-Management-System.git
```

### 2. Navigate into the project folder

```bash
cd Inventory-Management-System
```

### 3. Run the application

Windows

```bash
python main.py
```

macOS / Linux

```bash
python3 main.py
```
